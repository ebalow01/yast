# YieldMax ETF Analysis Dashboard

This directory contains the web dashboard for the YieldMax ETF analysis system.

## Files

- **`index.html`** - The main dashboard page (static HTML with embedded JavaScript)
- **`dashboard-old.html`** - The original React app entry point (preserved for React development)
- **`public/dashboard.html`** - The source dashboard file (same as index.html)
- **`public/data/`** - JSON data files generated by the analysis scripts
- **`src/`** - React/TypeScript source files for the advanced dashboard

## Usage

### Static Dashboard (Default)
Open `index.html` directly in a browser or serve via a web server. This loads data from `public/data/performance_data.json`.

### React Development Dashboard
To use the React version:
1. Rename `dashboard-old.html` back to `index.html`
2. Run `npm run dev`
